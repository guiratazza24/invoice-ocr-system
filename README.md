# 📄 Invoice OCR System

## 🚀 Overview
This project is a **smart OCR system** that extracts structured data from PDF invoices using Python.  
It converts unstructured invoice PDFs into clean JSON and Excel data.

---

## ✨ Features
- 📤 Upload PDF invoices via web interface (Streamlit)
- 🔍 OCR text extraction using Tesseract
- 🧠 Automatic invoice parsing (client, date, totals, VAT)
- 📊 Structured JSON output
- 📑 Table visualization
- 📥 Export to Excel

---

## 🛠️ Tech Stack
- Python
- Streamlit
- Tesseract OCR
- Pandas
- OpenPyXL
