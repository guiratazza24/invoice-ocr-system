# 📄 Invoice OCR System

## 🚀 Overview
This project is a smart OCR system that extracts structured data from PDF invoices using Python.  
It converts unstructured invoice PDFs into clean JSON and Excel formats using OCR and text processing.

---

## ✨ Features
- 📤 Upload PDF invoices via Streamlit web app
- 🔍 OCR text extraction using Tesseract
- 🧠 Automatic invoice parsing (client, date, totals, VAT, etc.)
- 📊 Structured JSON output
- 📑 Table visualization
- 📥 Export results to Excel

---

## 🛠️ Tech Stack
- Python
- Streamlit
- Tesseract OCR
- Pandas
- OpenPyXL
