# 📄 Invoice OCR System

## 🔍 Description
This project extracts structured data from PDF invoices using OCR and displays results in a web app.

## 🚀 Features
- Upload PDF invoices
- OCR text extraction
- Automatic invoice parsing
- JSON output
- Table view
- Excel export

## 🛠️ Tech Stack
- Python
- Streamlit
- Tesseract OCR
- Pandas

## ▶️ How to Run
```bash
pip install -r requirements.txt
streamlit run app.py
